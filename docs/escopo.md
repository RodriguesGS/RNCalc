# Escopo Geral RNCalc

## Objetivo

Uma calculadora simples que irá realizar as operações básicas da matemática `(+, -, *, /)`

## Escopo

- Interface com um input (entrada e saida)
- Teclado com números e operadores
- Realização de operações básicas
- Tratamento de erros
- Botões de controle como limpar tudo, apagar ultimo e calcular

## Tecnologias

- React Native (Expo)

## Critérios de entrega

- Operações básicas funcionando
- interface clara e com feedback para erros
- Código com teste unitários cobrindo todos cenários possiveis
